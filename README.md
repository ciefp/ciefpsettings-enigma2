## How to Install

```bash
# Download & extract
wget https://github.com/ciefp/ciefpsettings-enigma2/releases/download/latest/ciefp-E2-5sat-19E-16E-13E-1.9E-0.8W.tar.gz
tar -xzf ciefp-E2-*.tar.gz -C /etc/enigma2/

# Restart GUI
init 4 && sleep 5 && init 3
```
